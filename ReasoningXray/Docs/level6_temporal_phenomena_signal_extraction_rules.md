1️⃣ Stabilization Phase — Detection Rules
Primary Structural Pattern

trajectory path = narrowing OR workingDiagnosis

certainty trend = stable OR mildly increasing

reasoning movement = low change density

turning points = absent

visit spacing = moderate or widening

Interpretation Logic

Reasoning is holding hypothesis under time observation.

Exclusion Conditions

Do NOT classify stabilization if:

momentum = retreating

epistemic status = volatile

turning point just occurred

2️⃣ Narrowing Phase — Detection Rules
Primary Structural Pattern

trajectory path = narrowing

certainty trend = unclear → increasing

reasoning change density = moderate

epistemic status = cautious but directional

Interpretation Logic

Reasoning field is contracting toward focus.

Supporting Signals

repeat visits with incremental refinement

uncertainty still active

3️⃣ Oscillation Phase — Detection Rules
Primary Structural Pattern

trajectory path fluctuates across visits

certainty trend = alternating or unclear

reasoning change density = high

turning points = recurring or near

Epistemic Signature

trust signal = mixed / recalibrating

momentum = inconsistent

Interpretation Logic

Reasoning is actively comparing models.

4️⃣ Consolidation Phase — Detection Rules
Primary Structural Pattern

trajectory path = workingDiagnosis OR confirmation

certainty trend = increasing

reasoning movement = low

epistemic status = stabilizing

uncertainty flag = reducing

Timeline Signature

visit spacing becomes structured

plan language consistency increases

Interpretation Logic

Working explanation is gaining structural coherence.

5️⃣ Monitoring Plateau — Detection Rules
Primary Structural Pattern

trajectory path unchanged across multiple visits

reasoning movement minimal

certainty trend flat

uncertainty still present

visit spacing widening

Epistemic Signature

trust signal = steady

volatility low

Interpretation Logic

Time itself is being used as diagnostic instrument.

6️⃣ Acceleration Window — Detection Rules
Primary Structural Pattern

turning point detected

reasoning change density spike

trajectory path shifts toward confirmation or new narrowing

certainty trend = sharp increase

Timeline Signature

visits cluster

multiple decisions compressed

Interpretation Logic

New decisive information has entered reasoning field.

7️⃣ Re-Exploration Loop — Detection Rules
Primary Structural Pattern

trajectory path moves backward in structural stage
(e.g., workingDiagnosis → narrowing OR exploration)

turning point present

epistemic status = recalibrating

uncertainty increases

Interpretation Logic

Reasoning model reset — but at higher informational maturity.

⚙️ Dominance Resolution Rule (Very Important)

Multiple temporal phenomena may be detectable simultaneously.

Level 6 must:

select one dominant temporal phenomenon

based on strongest structural coherence signal

not based on emotional weight

Hierarchy of dominance (default):

Acceleration Window

Re-Exploration Loop

Oscillation

Consolidation

Narrowing

Stabilization

Monitoring Plateau

This prevents ambiguous temporal messaging.

🧭 Temporal Persistence Rule

Phenomenon should persist across at least two visits
before being surfaced — unless:

turning point detected

acceleration window signal spike

This avoids temporal noise.

✅ Level 6 Detection Success Criteria

No medical content interpretation required

Deterministic rule set possible

Works purely on trajectory + epistemic semantics

Produces stable temporal labeling

Avoids over-reactivity to single visit fluctuations
