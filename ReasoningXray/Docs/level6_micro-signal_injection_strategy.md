🕰️ ReasoningXray Level 6
Temporal Micro-Signal Injection Strategy (Scope Charter)
1. Governing Principle

Temporal signals must feel:

ambient, optional, and secondary to reasoning understanding.

They must never compete with:

trajectory meaning

trust status

doctor-said structure

watchpoints

They are temporal atmosphere, not content.

2. Primary Injection Zone
✅ Trajectory Panel — Terminal Soft Caption

Location

bottom edge of the existing trajectory presentation panel

after “What to Expect Next”

visually lighter than all prior text

Rationale

This position:

preserves reasoning comprehension hierarchy

introduces time-sense only after meaning is understood

feels like a reflective afterthought, not instruction

Behavior

appears only when temporal dominance stable

max one signal

fades or rotates across visits

no icon required

Visual Weight

smaller font

softer color (secondary text style)

increased line spacing

This creates:

Perceptual quietness.

3. Secondary Injection Zone
✅ Timeline Visit Node — Micro Tempo Hint (Optional)

Location

subtle caption under a visit node

only when a temporal shift occurs (e.g., acceleration window)

Examples of Use

sudden clustering of visits

turning point detected

stage regression

Rationale

This ties tempo awareness directly to visit spacing.

Important:
Must not appear on every node.
Must not form a pattern users try to “track.”

4. Tertiary Injection Zone
✅ Trust Badge Contextual Hover (Advanced / Later)

Only if user interacts.

Possible behavior:

hover / tap reveals one temporal sentence

integrates trust + time perception

But this is:

not required for Level 6 MVP

should be deferred unless UX testing demands

5. Injection Frequency Governance
Hard Limits

maximum one temporal signal per visit rendering

minimum two-visit persistence before signal change

no signal if trajectory volatility high

no signal if epistemic status already conveys instability

Temporal signals must stabilize perception, not amplify noise.

6. Signal Rotation Logic

To avoid habituation:

same phenomenon → rotate phrasing

different phenomenon → reset phrasing set

do not repeat identical signal in consecutive renders

But:

avoid feeling “dynamic” or gamified

This is slow semantic drift, not animation.

7. Silence Rule (Very Important)

Level 6 must sometimes say nothing.

No temporal signal should appear when:

trajectory clarity already high

confirmation phase stable

case closed / monitoring minimal

epistemic trust strong and calm

Silence reinforces:

temporal maturity.

8. Anti-Authority Guardrail

Temporal signals must never feel like system knowledge of the future.

Avoid:

progress metaphors (journey milestones)

improvement metaphors (getting better)

resolution metaphors (nearing answer)

countdown metaphors (soon / next step)

They must feel like:

observations about reasoning tempo in general.

9. Success Experience

When injection is correct:

Patients do not consciously think:

“The app is guiding me.”

Instead they feel:

time makes more sense

quiet periods feel less threatening

reversals feel less chaotic

follow-ups feel intentional

The emotional tone becomes:

structured patience.
