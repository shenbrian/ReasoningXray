# Level 3 Stress Test Notes

## Scenario 1 — Straight confirmation path
Expected:
- Path: confirmation
- Certainty: increasing
- Momentum: advancing
- Uncertainty: false

## Scenario 2 — Exploration that never settles
Expected:
- Path: earlyExploration or narrowing
- Certainty: unclear
- Momentum: paused or looping
- Uncertainty: true

## Scenario 3 — Reopened case
Expected:
- Path: reopened
- Certainty: stable or decreasing
- Momentum: redirected or advancing
- Uncertainty: true

## Scenario 4 — Monitoring loop
Expected:
- Path: monitoring
- Certainty: stable or unclear
- Momentum: looping
- Uncertainty: true

## Scenario 5 — Fast escalation without contradiction
Expected:
- Path: narrowing or confirmation
- Certainty: increasing
- Momentum: advancing
- Uncertainty: false or borderline true
